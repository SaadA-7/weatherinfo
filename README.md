## Setup Instructions

### Local Development

1. Install dependencies:
  ```bash
  npm install
  ```

2. Configure environment variables in `server.js`:
  - Uncomment: `require('dotenv').config();`

3. Start the application:
  ```bash
  npm start
  ```
